创新创业训练营-Day1

发表于 2017-04-19 | 分类于 产品经理 | | 阅读次数

上周六参加了学校的创新创业训练营, 现把其中的收获记录如下

  • ###开营仪式致辞总结:
    • 心态:做人要帅,心态上要帅, 可以理解为一种雷厉风行的作风,也可以是一种不骄不躁的淡定,心态上帅是指一种有人格魅力的优秀心态
      • 事业:事业上要有绝对的竞争壁垒,使自己的产品具有一种核心竞争力, 最重要的是具有雷厉风行的执行力
      • 合格的创业项目:
        • 能够做到快速的成本转移
        • 现金流足够的安全
        • 业务综合能力

  • ###现代商业理念与好顾客管理( 胜出红海 )

    • 现代商业理念
      • 爱
        • 爱可以理解为一种博爱, 可以是一种能发现并想解决生活中给人带来不便利问题的思维, 可以是一种愿意为人类创造价值的精神导向;
        • 在拥有这种博爱之后, 我们能逃避生活所带来的麻木, 更多去发现生活的不便利, 也即是被赋予了一种发现商机的眼睛;
        • 而博爱则要求我们更多的去创造价值, 是一种劳动和价值的输出, 让我们的项目更具有持久发展力;
      • 信
      • 商
        • 商是创业的手段,是一种博弈., 是一种己所欲,并且施于人的商业手段, 敢于说话, 敢于失败, 做有意义的冒险
      • 赢
        • 赢可以说是商业最终的成就, 而在这里拆字进行解读赢的精神;
        • 可以理解为成功是以自己的健康身体为基础进行的冒险,首先具有一个可持续发展的身体.
        • 同样,成功离不开一颗平常心; 这里强调的是心态,冒险过程中难免会失败,要以一颗平常心来对待
        • 金钱作为一种资源, 毫无疑问可以支撑我们的冒险, 但相对前两者而言就不是那么重要了, 因为你首先了解的创造的是价值, 而不是大把大把花不完拿着压手的金钱
        • 而中间的口就意味着我们要敢说, 利言, 交流是沟通的唯一途径
        • 亡则要求我们要有一种危机意识, 居安思危, 时刻保持反思; 同样也对我们的创业竞争形成了一种要求——不求挤死对手, 而是要在收支平衡中获得生存

    • 好顾客管理

      • 3w法则

        • 为谁服务( whom )?
          精准定位, 找准目标客户群体; 自己以为在有精准客户后能更加细分自己的产品,实现精准的定位营销和服务, 拥有一批忠诚的客户群体, 这将会使自己的产品具有一种核心竞争力: 例如今天下午的商业观察中, 19tea这家店铺就是因为细分了客户为喜欢欧式面包的年轻人而一把紧抓果茶和欧式面包,从而使自己的产品具有一种核心竞争力, 即使是在刚开业不到半个月顾客就踏破门庭
        • 顾客需要我们的什么( what )?
          顾客需要的并不是简单的产品, 而是与其他产品不同的地方, 例如空调, 顾客需要的并不是单纯的调节温度等能力, 他们需要一些不同的的亮点之处,例如空调的外观, 更多其他产品不具有的健康功能 ,这也怎好和我们的创新相吻合, 这也是我们产品核心竞争力的关键所在
        • 而谁能满足顾客们的需求( who )?

      • 3暨法则

        • 滚动法则
          熟悉理财的朋友都清楚, 理财做的是让钱生钱的过程, 而我们[平时的商业策略中这一点则极为重要, 钱生钱产生的是一种指数效应, 前景是非常可观的, 过程中肯定不会像张院长所说的那样指数爆炸, 但至少能让我们的投入成本持续增加, 而收益相对来说就会持续增长; 但这里和前面开营致辞是矛盾的, 持续投入成本就意味着我们的现金流不是很完善, 所以我们要把持好其中的平衡, 在边际效应下合理的投入成本, 努力实现钱滚钱的滚动效应
        • 节约定律
          这里的节约可以是个人的习惯, 但更多的是优化产业链, 降低成本
        • 权钱交易
          自古就有小权生大钱的现象, 并且很普遍
          学会将应该设权设置成产权
          将没有设权的资源设置成产权
          将不能出售的权利变成能够出售的权利:股权, 债权
  • ###商业观察总结
    • 在这一次商业观察中我们是没有很好抓住热点的. 和其他队伍相比较, 我们在这一路上就只是在太阳新天地这个小地方转悠, 没有抓住热点
    • 商业观察后的分析没有很深入的进行,可能是因为队员基本都是理科生,对于商业的角度拿捏得不是很准确,导致分析不够深入;
    • 但这只是一部分原因,更多的需要分析为什么商业眼光不及他人?平时信息了解的比较少?平时缺乏自己的分析见解只是一味迷信别人的分析却没有自己真正的重新再分析?我想平时自己对于商业缺乏思考才是这次做的不够好的根本原因.
    • 就辅助条件做得不够好:例如ppt, 没有一种商务风范,排版不够合理, 尽管ppt讲究的是简洁明了,但是关键点并没有说得特别清楚
    • 观察也仅仅是停留在分析层面上: 并没有提出实际可行的改良方案, 这是一种批判性思维欠缺的表现
    • 演讲思维不够清晰, 不能把演讲的条理分特别清楚, 相比别的队伍, 有人以演讲慷慨打感情牌取胜, 有的则演讲不够搞笑来凑风格占据上位, 而我们队并没有太明显的特色, 这个是需要在短时间之内快速磨合的
    • 不是抱着绝对全力以赴的态度去做事的, 略懒散: (
